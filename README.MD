# Anomaly

> **Autonomous task execution engine with built-in versioning, reliability tracking, and intelligent self-optimization.**

A lightweight but powerful system for managing, executing, and optimizing tasks across Python, Bash, and Node.js. Perfect for DevOps automation, job scheduling, monitoring workflows, or anywhere you need tasks to run reliably and learn from themselves.

## Features

- 🔄 **Subprocess Isolation** — Every execution runs in its own process; crashes can't touch the operator
- 📊 **Reliability Tracking** — Success/failure rates, durations, and error history per task
- 🔍 **Smart Duplicate Detection** — Finds tasks that do the same thing, even with different code
- 🗑️ **Auto-Optimization** — Automatically deprecates stale, flaky, or redundant tasks
- 📦 **Full Versioning** — Every update preserved; roll back anytime without data loss
- 🔗 **Dependency Graphs** — Tasks
